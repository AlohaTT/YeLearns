###夜月咨询

夜月：一款符合Google Material Design的Android客户端。项目使用MVPArms作为底层框架，然后自己扩展了一个UI框架YeLibrary，实现了各种常用的列表页，Tab页等等，便于快速开发应用。现主要包括：玩安卓，DIYcode，干货，微信资讯，游明星空，影视，壁纸等多个模块。
